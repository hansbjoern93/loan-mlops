# Loan MLOps Project

## Voraussetzungen

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installiert
- [VS Code](https://code.visualstudio.com/) installiert
- VS-Code-Erweiterung **Dev Containers**
- VS-Code-Erweiterungen **Python** und **Jupyter** (werden im Dev Container automatisch mitgeladen)

---

## Repository klonen

```bash
git clone https://github.com/hansbjoern93/loan-mlops.git
cd loan-mlops
```
